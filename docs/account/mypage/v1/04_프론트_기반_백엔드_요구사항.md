## 3. 완료된 API 명세

### 3.1. ✅ 비밀번호 변경 (이중 인증)

**엔드포인트**: `PUT /api/mypage/password`

**인증 흐름**:
1. `POST /api/auth/verify-password` - 현재 비밀번호 확인
2. `POST /api/auth/email/send-change-code` - 이메일 인증 코드 발송
3. `PUT /api/mypage/password` - 새 비밀번호 설정

**Request Body**:
```json
{
  "code": "123456",
  "newPassword": "newPassword123!"
}
```

### 3.2. ✅ 회원 탈퇴

**엔드포인트**: `DELETE /api/mypage/account`
- Soft Delete 처리
- 응답: `204 No Content`

## 4. 필요한 API 명세 (Phase 3)

### 4.1. 활동 통계 조회
- **URL**: `/api/mypage/stats`
- **Method**: `GET`
- **Response**:
  ```json
  {
    "postCount": 15,
    "commentCount": 42,
    "bookmarkCount": 8,
    "totalViews": 1250
  }
  ```

### 4.2. 내 게시글 목록 조회
- **URL**: `/api/mypage/posts`
- **Method**: `GET`
- **Query Params**: `page=0`, `size=5`
- **Response**:
  ```json
  {
    "content": [
      {
        "id": 101,
        "title": "스프링 부트 질문있습니다",
        "category": "Q&A",
        "views": 45,
        "commentCount": 2,
        "createdAt": "2025-11-28T10:00:00"
      }
    ],
    "totalPages": 3,
    "totalElements": 15
  }
  ```

### 4.3. 내 댓글 목록 조회
- **URL**: `/api/mypage/comments`
- **Method**: `GET`
- **Query Params**: `page=0`, `size=5`
