## 🔍 현재 상태

### 문제점
```tsx
// CommunityWritePage.tsx
const [category, setCategory] = useState<BoardCategory>('QUESTION');
```

- 글쓰기 페이지 진입 시 **항상 "문의사항"이 기본 선택**
- 사용자가 "코딩이야기" 탭에서 글쓰기 버튼을 눌러도 "문의사항"으로 설정됨
- 매번 수동으로 카테고리를 변경해야 함

---

## ✅ 요구사항

| 사용자 동작 | 기대 결과 |
|------------|----------|
| "전체" 탭에서 글쓰기 클릭 | 기본 카테고리 (코딩이야기) 선택 |
| "공지사항" 탭에서 글쓰기 클릭 | 공지사항 선택 (관리자만 가능) |
| "문의사항" 탭에서 글쓰기 클릭 | 문의사항 선택 |
| "진로이야기" 탭에서 글쓰기 클릭 | 진로이야기 선택 |
| "코딩이야기" 탭에서 글쓰기 클릭 | 코딩이야기 선택 |

## 🔧 구현 방안

### 방법 1: URL Query Parameter 사용 (추천 ⭐)

#### 장점
- URL에 상태가 보여서 공유/북마크 가능
- 새로고침해도 유지됨
- 직관적이고 디버깅 쉬움

#### CommunityPage.tsx 수정
```tsx
// 기존
<Link to="/community/write" className="btn-primary flex items-center gap-2">

// 변경
<Link 
    to={`/community/write${currentCategory ? `?category=${currentCategory}` : ''}`}
    className="btn-primary flex items-center gap-2"
>
```

#### CommunityWritePage.tsx 수정
```tsx
import { useSearchParams } from 'react-router-dom';

const CommunityWritePage = () => {
    const [searchParams] = useSearchParams();
    
    // URL에서 카테고리 읽기, 없으면 기본값 'CODING_STORY'
    const initialCategory = (searchParams.get('category') as BoardCategory) || 'CODING_STORY';
    
    const [category, setCategory] = useState<BoardCategory>(initialCategory);
    // ...
};
```

#### URL 예시
| 현재 탭 | 글쓰기 URL |
|--------|-----------|
| 전체 | `/community/write` |
| 문의사항 | `/community/write?category=QUESTION` |
| 코딩이야기 | `/community/write?category=CODING_STORY` |

---

### 방법 2: React Router state 사용

#### CommunityPage.tsx
```tsx
<Link to="/community/write" state={{ category: currentCategory }}>
```

#### CommunityWritePage.tsx
```tsx
import { useLocation } from 'react-router-dom';

const location = useLocation();
const initialCategory = (location.state?.category as BoardCategory) || 'CODING_STORY';
```

#### 단점
- URL에 상태가 보이지 않음
- 새로고침 시 상태 유실
- 직접 URL 접근 시 기본값 사용

## ⚠️ 고려사항

1. **공지사항 카테고리**: 관리자만 선택 가능하도록 권한 체크 필요
2. **잘못된 카테고리**: URL에 유효하지 않은 카테고리 입력 시 기본값 사용
3. **기본값 변경**: 현재 `QUESTION` → `CODING_STORY`로 변경 권장 (가장 일반적인 게시글)

---

## 🧪 테스트 시나리오

1. 각 탭에서 글쓰기 버튼 클릭 → 해당 카테고리 자동 선택 확인
2. URL 직접 접근 (`/community/write?category=COURSE_STORY`) → 진로이야기 선택 확인
3. 잘못된 URL (`/community/write?category=INVALID`) → 기본값(코딩이야기) 선택 확인
4. 새로고침 → 카테고리 유지 확인
