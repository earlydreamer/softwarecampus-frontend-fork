# 메인 배너 타이틀 줄바꿈 수정 보고서

## 1. 문제 상황
- **증상**: 데스크탑(풀사이즈) 화면에서 배너 타이틀("재직자를 위한 직무 향상 과정")이 불필요하게 줄바꿈됨. 특히 "과\n정"처럼 단어 중간이 끊기는 현상 발생.
- **원인**:
  1. **너비 제한**: 텍스트 컨테이너의 최대 너비가 `max-w-2xl` (약 672px)로 고정되어 있어, 폰트 크기가 커지는 데스크탑 화면(`text-6xl`)에서는 공간이 부족함.
  2. **줄바꿈 규칙**: 기본 줄바꿈 규칙(`word-break: normal`)이 적용되어 있어, 너비가 부족하면 글자 단위로 줄바꿈이 발생할 수 있음(브라우저/언어 설정에 따라 다름).

## 2. 해결 방법
- **너비 제한 완화 (Responsive Max-Width)**:
  - 모바일은 기존 `max-w-2xl` 유지.
  - 데스크탑(`md` 이상)에서는 `max-w-4xl` (약 896px), 대형 화면(`lg` 이상)에서는 `max-w-5xl` (약 1024px)로 너비 제한을 대폭 늘림.
  
- **단어 단위 줄바꿈 (`break-keep`)**:
  - `break-keep` (CSS `word-break: keep-all`) 클래스를 추가하여, 줄바꿈이 필요하더라도 단어 중간이 아닌 공백(어절) 단위로 끊기도록 강제함.

## 3. 변경 코드 (`src/components/home/HeroBanner.tsx`)

```tsx
// 변경 전
<div className="max-w-2xl ...">
    <h2 className="...">...</h2>
</div>

// 변경 후
<div className="max-w-2xl md:max-w-4xl lg:max-w-5xl ...">
    <h2 className="... break-keep">...</h2>
</div>
```

## 4. 결과
- 데스크탑 화면에서 긴 타이틀도 한 줄(또는 자연스러운 두 줄)로 시원하게 표시됨.
- "과\n정"과 같이 글자가 잘리는 현상이 사라지고 가독성이 향상됨.
