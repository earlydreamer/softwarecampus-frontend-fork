# 커뮤니티 프론트엔드 현황 분석

## 1. 파일 구조

```
src/
├── pages/
│   ├── CommunityPage.tsx          # 게시글 목록 페이지
│   ├── CommunityDetailPage.tsx    # 게시글 상세 페이지
│   ├── CommunityWritePage.tsx     # 게시글 작성 페이지
│   └── CommunityEditPage.tsx      # 게시글 수정 페이지
├── services/
│   ├── communityService.ts        # API 호출 서비스
│   └── api/types.ts               # API 응답 타입 정의
├── components/
│   ├── home/CommunitySection.tsx  # 홈 페이지 커뮤니티 섹션
│   └── editor/TiptapEditor.tsx    # 리치 텍스트 에디터
└── types/index.ts                 # Board, Comment 타입 정의
```

## 2. 페이지별 기능

### 2.1 CommunityPage.tsx (게시글 목록)
- **카테고리 필터**: NOTICE, QUESTION, COURSE_STORY, CODING_STORY
- **정렬**: 최신순, 추천순, 조회순, 댓글순
- **검색**: 전체, 제목, 내용, 제목+내용, 글쓴이, 댓글
- **페이징**: URL 쿼리 파라미터 기반
- **반응형**: 데스크탑(테이블 형식) / 모바일(카드 형식)

### 2.2 CommunityDetailPage.tsx (게시글 상세)
- 게시글 조회 (제목, 본문, 작성자, 조회수, 추천수)
- 추천 기능 (중복 추천 방지)
- 댓글 CRUD (작성, 수정, 삭제)
- 작성자 본인 확인 → 수정 버튼 표시

### 2.3 CommunityWritePage.tsx (게시글 작성)
- 카테고리 선택
- 제목 입력 (최대 255자)
- 본문 입력 (Tiptap 리치 에디터, 최소 10자)
- 작성 취소 시 확인 모달
- 브라우저 새로고침 시 경고

### 2.4 CommunityEditPage.tsx (게시글 수정)
- 기존 게시글 데이터 로드
- 수정 후 저장

## 3. 타입 정의

### 3.1 프론트엔드 타입 (`types/index.ts`)

```typescript
export type BoardCategory = 'NOTICE' | 'QUESTION' | 'COURSE_STORY' | 'CODING_STORY';

export interface Board {
    id: number;
    title: string;
    text: string;
    category: BoardCategory;
    account: {
        id: number;
        userName: string;
        avatar?: string;
    };
    hits: number;
    secret: boolean;
    createdAt: string;
    updatedAt?: string;
    likeCount: number;
    like?: boolean;      // 현재 사용자가 추천했는지
    owner?: boolean;     // 현재 사용자가 작성자인지
    commentCount?: number;
    hasAttachment?: boolean;
    comments?: Comment[];
}

export interface Comment {
    id: number;
    boardId?: number;
    text: string;
    account: {
        id: number;
        userName: string;
        avatar?: string;
    };
    createdAt: string;
    updatedAt?: string;
    isDeleted?: boolean;
    subComments?: Comment[];
}
```

## 4. 현재 구현된 API 호출 (`communityService.ts`)

| 함수명 | 메서드 | 엔드포인트 | 설명 |
|--------|--------|-----------|------|
| `fetchBoardPosts` | GET | `/api/boards` | 게시글 목록 조회 |
| `fetchBoardPost` | GET | `/api/boards/{id}` | 게시글 상세 조회 |
| `createBoardPost` | POST | `/api/boards` | 게시글 작성 |
| `recommendBoardPost` | POST | `/api/boards/{id}/recommends` | 게시글 추천 |
| `createComment` | POST | `/api/boards/{id}/comments` | 댓글 작성 |
| `updateComment` | PATCH | `/api/boards/{id}/comments/{commentId}` | 댓글 수정 |
| `deleteComment` | DELETE | `/api/boards/{id}/comments/{commentId}` | 댓글 삭제 |
