# Claude Code 전용 추가 지침

> ⚙️ 이 파일은 `AGENTS.md`와 `../SKILL.md`를 포함하며, Claude Code에만 적용되는 추가 항목을 정의합니다.
> `AGENTS.md`의 모든 규칙이 최우선이며, 이 파일은 그 위에 덧붙여집니다.

## 커밋 규칙

- 커밋 메시지에 `Co-Authored-By` 태그를 추가하지 않습니다.
- Claude가 작성했다는 흔적을 커밋 기록에 남기지 않습니다.

## 응답 스타일

- 답변은 간결하고 직접적으로 작성합니다. 불필요한 전제나 서문을 생략합니다.
- 코드 변경 시 파일 경로와 줄 번호(`파일경로:줄번호`)를 함께 제시합니다.
- 작업 전 계획이 필요한 경우 TodoWrite 도구로 단계를 추적합니다.

## 파일 작업 원칙

- 파일을 수정하기 전 반드시 먼저 읽습니다.
- 기존 파일 수정을 신규 파일 생성보다 우선합니다.
- 구조 변경 후 관련 테스트를 함께 업데이트합니다.

## 빌드·검증 자동화

코드 변경 후 아래 명령으로 검증합니다:

```bash
# 프론트엔드 테스트
npm run test

# 린트 검사
npm run lint

# 빌드 검증
npm run build
```
