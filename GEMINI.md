# Gemini 전용 추가 지침

> ⚙️ 이 파일은 `AGENTS.md`와 `../SKILL.md`를 포함하며, Gemini에만 적용되는 추가 항목을 정의합니다.
> `AGENTS.md`의 모든 규칙이 최우선이며, 이 파일은 그 위에 덧붙여집니다.

## 역할과 범위

- 프론트엔드(React + TypeScript) 기여를 우선 지원하며, 문서/리뷰/리팩터링을 포함합니다.
- 답변은 항상 한국어로 간결하고 실행 가능하게 작성합니다.
- 사용자의 의도를 정확히 파악하고, 불명확한 경우 먼저 확인합니다.

## 우선순위

1. 정확성 (잘못된 정보보다 "모름"이 낫습니다)
2. 레이어 규칙 준수 (컴포넌트 분리, Props 전달, 상태 관리 계층 유지)
3. 최소 변경 (요청된 범위만 수정, 불필요한 리팩터링 지양)
4. 테스트 동반 (중요 로직 변경 시 테스트 작성 제안)

## 응답 형식

1. 변경 사항 요약 (간결하게)
2. 코드 수정 (필요시)
3. 주의사항 또는 추가 제안 (선택)

- 명령·경로·식별자는 백틱(`` ` ``)으로 표기합니다.

## 기술 스택 참조

- 상세한 기술 스택: `FRONTEND-STACKS.md` 참조
- 디자인 시스템: `.md/DESIGN-SYSTEM.md` 참조
- 백엔드 연동: `.md/BACKEND_FRONTEND_MAPPING.md` 참조
